# Como usar:

Primero clona el repositorio
```sh
git clone https://github.com/Onono2008/PYcrypt
```
Luego entra al directorio del script
```sh
cd PYcript
```
Por ultimo, ejecuta el script mediante el comando
```sh
python3 pycript.py
```


# Uso:

``sh
e -> Encriptar
``
``sh
d -> Desencriptar
``
``sh
o -> Salir
``
